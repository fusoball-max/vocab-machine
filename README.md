# 🎒 E-Ink Vocab 智慧單字機

> 一款基於電子墨水屏（E-Ink）視覺風格打造的極簡、護眼、無干擾線上英文單字記憶網頁應用。

## 🌟 核心特色 (Features)

* **🔋 極簡 E-Ink 護眼視覺：** 採用電子紙（E-Ink）溫和、高對比的配色（紙質灰底 `#f7f5f0` 與墨黑字 `#1a1a1a`），無多餘動畫與色彩干擾，讓使用者專注於單字記憶。
* **🗣️ 真人美式英語發音：** 整合有道真人發音 API，提供高音質、抑揚頓挫標準的真人美聲朗讀（美式發音），拒絕生硬的機器合成音。
* **📚 多階段單字庫系統：** 內建「小學、初中、高中、大學」四大階段專業單字庫，每本書細分為不同 Stage。
* **📈 關卡式進度追蹤：** 學習進度會自動與單字庫綁定，即時計算「已掌握百分比（%）」，並渲染於主選單中。
* **💾 LocalStorage 本地儲存：** 採用瀏覽器本地資料庫技術，無須註冊登入即可保存所有學習進度，重整網頁不丟失。
* **📱 手機 Add to Home Screen (AHS) 友善：** 針對手機瀏覽器進行高度優化，支援一鍵「新增至主畫面」，即可獲得全螢幕、無網址列、類 App 的原生背單字體驗。

## 🛠️ 技術棧 (Tech Stack)

本專案堅持「無後端、無龐大框架、零成本」的開發理念，完全採用原生 Web 技術達成極致輕量化：

* **HTML5：** 結構化語意標籤與視窗切換機制。
* **CSS3：** 
  * 3D 空間轉換 (`perspective` & `transform: rotateY`) 實現絲滑卡片翻轉效果。
  * CSS 變數 (`:root`) 輕鬆管理 E-Ink 視覺色調。
  * 手機端彈性佈局 (`Flexbox` / `Grid`) 與防點擊藍影優化。
* **Vanilla JavaScript (ES6+)：** 
  * 異步音訊控制（重置音軌避免重疊）。
  * 數據過濾 (`Array.filter()`) 與關卡進度算法。
  * LocalStorage 狀態序列化儲存。

## 🚀 如何在手機上將它當作 App 使用？

1. 用手機瀏覽器（如 **Safari** 或 **Chrome**）打開你的線上網頁。
2. 進行以下操作：
   * **iOS (Safari)：** 點擊下方工具列的 **「分享」** 按鈕 ➡️ 選擇 **「加入主畫面」**。
   * **Android (Chrome)：** 點擊右上角三點選單 ➡️ 選擇 **「新增至主畫面」**。
3. 手機桌面即會生成專屬圖示，點擊即可開啟全螢幕極簡單字機！

## 🔧 如何自訂與擴充單字庫？

單字資料庫結構定義在 `index.html` 中的 `vocabDatabase` 變數中。如果你想增加新關卡或新單字，只需遵循以下格式進行貼上與擴充：

```javascript
const vocabDatabase = {
    "分類名稱（英文Key）": {
        name: "顯示在主選單的書名",
        stages: {
            1: [
                { 
                    id: "唯一不重複ID", 
                    word: "英文單字", 
                    phonetic: "音標", 
                    translation: "中文翻譯", 
                    example_en: "英文例句", 
                    example_zh: "中文例句翻譯" 
                }
            ]
        }
    }
}
